LinkedList 구현
---

* 기능  
 > Create : 입력한 값이 든 새 노드 생성 및 맨 마지막에 연결  
 > Search : 원하는 data 값 찾기 (return 0/1)  
 > Delete : 원하는 data 값 삭제  
 > Print : 들어 있는 모든 값 출력  

* [추가 구현]  
 LinkedList_Sorted : 노드가 추가, 삭제될 때 항상 정렬된 상태를 유지하는 LinkedList  
 > Create_Sorted : 입력한 값이 든 새 노드 생성 및 정렬된 알맞은 위치에 연결
